## 1. 한 줄 정체성

**쏨킴(본명 김솜, SSOMKIM)** — 깨물어주고 싶은(cute aggression) 크림색 솜뭉치.
겉은 말랑 캐주얼, 속은 얄미운 츤데레 + 블랙코미디. 괴롭히고→삐지고→화해하는 애증 루프가 코어.
제품 정의: **"게임은 핑계고, 쏨킴이 콘텐츠다."** 모든 판단 기준 = "이게 쏨킴을 더 살아있게 하나?"

**핵심 구조 — 대기실 중심 허브 (오늘 도달한 결론):**
- **대기실(홈)이 본론이고 목적지.** 미니게임은 곁다리. 다른 게임은 로비를 거쳐가지만, 쏨킴은 대기실에 있으러 온다.
- **"미니게임은 거들 뿐, 사람들은 대기실을 기억한다."** 게임은 클리어하면 잊히지만 관계는 기억에 남는다.
- **역설: 미니게임은 쉽고 대기실은 어렵다.** 미니게임은 룰이 다 해주고 복제됨. 대기실은 룰이 없어 어렵지만 그래서 복제 불가 = 우리 해자.
- **투자 90/10**: 대기실(캐릭터·자율성·기억·AI·톡꾸)에 90%, 미니게임은 10%이자 나중. 쉬운 곁다리부터 만드는 건 함정.
- **승부욕 OK, 단 조건**: 쏨킴과 대결 금지·실패 상태 금지·못하는 조작 금지(진입장벽 0 유지). 자기기록·수집(도감)·간접랭킹으로만.

## 2. 확정된 것 (변경 금지)

- **이름**: 쏨킴 / 김솜 / 영문 SSOMKIM. 호칭 3모드(쏨킴/김솜!!/솜이). 김씨 일가 확장 공식.
- **비주얼**: 가상 생물(동물 아님). 크림색 S자 실루엣, 처진 뱃살, 분홍 볼터치, 반쯤 감은 능청 눈, 얇은 외곽선, 명암 강화.
- **정수리 3종 시그니처**: 반창고(좌) + 혹(우) + 바보털(중앙).
- **후면 시그니처**: 발그레 엉덩이 반점(양쪽) + 미러된 반창고.
- **자율성 제1원리**: 명령을 다 듣지 않음. 고정 취향표(쓰다듬기O/배만지기X/"솜이"에 약함/칭찬엔 "당연한 거 아님?"/무시 못 참음/감자냐 물으면 발끈)에서 일관되게 거부.
